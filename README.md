# Women Legal Assistant ⚖️

An AI-powered web app that helps users find relevant Indian laws and actions based on their problems using semantic search.

## Features

* Semantic search using embeddings
* Displays relevant law and actions
* Simple web interface
* Fast search with FAISS

## Tech Stack

Python, Flask, FAISS, Sentence Transformers, HTML

## How to Run

1. Create virtual environment
2. Install dependencies
3. Run `python embeddings/build_index.py`
4. Run `python app.py`
